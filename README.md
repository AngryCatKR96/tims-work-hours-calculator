# TIMS 근무시간 계산기

티맥스 TIMS 일일근태조회 페이지에서 이번 주(월~금)의 출퇴근 시간을 기준으로 초과 근무 시간을 계산하는 크롬 익스텐션입니다.

## 기능

- 이번 주 월~금 평일의 근무 시간 자동 계산
- 출퇴근 시간 기준 초과/부족 시간 표시 (+/-)
- 미래 날짜(아직 오지 않은 날)에 대한 기본값 설정 (9:00~18:00)
- 사용자가 미래 날짜의 출퇴근 시간을 직접 입력/수정 가능
- 입력한 값은 자동으로 저장되어 다음 접속 시에도 유지
- 근태 표에 초과근무 컬럼으로 임베딩 표시
- 주간 총계 자동 계산

<!-- 설치 방법은 Chrome 웹 스토어 등록을 전제로 생략합니다 -->

## 사용 방법

1. TIMS 일일근태조회 페이지에 접속
2. 익스텐션이 자동으로 근태 표에 "초과근무" 컬럼을 추가합니다
3. 각 날짜별로 초과/부족 시간이 표시됩니다
4. 미래 날짜의 경우 출퇴근 시간을 직접 입력할 수 있습니다

## 기술 스택

- Chrome Extension Manifest V3
- Vanilla JavaScript
- Chrome Storage API

## 주의사항

- TIMS 페이지의 frameset 구조를 지원합니다
- API를 통해 근태 데이터를 가져옵니다
- 세션 쿠키를 자동으로 사용하여 인증합니다

## 개인정보처리방침
사용자의 웹사이트 콘텐츠 형태 (HTML 요소)를 확인하지만 수집과 사용할 계획이 없습니다.
